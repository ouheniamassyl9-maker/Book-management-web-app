# Book-management-web-app
A simple web application for managing a book collection.
## Features
- User registration and login system
- Add, list, edit, and delete books
- Client-side data storage using localStorage

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)

## Current Status
Educational project demonstrating front-end fundamentals. Uses localStorage for authentication and data storage (demonstration purposes only - not production-ready).

## Future Plans
- Integration with actual PDF book links
- Backend database implementation
